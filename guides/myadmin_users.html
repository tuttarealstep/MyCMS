Users

Users can login or register to the website only if the theme permit this.

Post menu

When you click on the users menu in the top left of the topbar, you can view three more options.

  • Users Permissions: this option redirect you to a page, for give some permission to the users.
  • Bans users: this menu redirect you to a page for ban user from the website.
  • Info users: this menu redirect you to a page for view the users info.


Users Permissions

On this page you can easily add, remove special permission to any user.
Permissions are a type of grade on your website.

  • Administrator (3 Grade), can access everywhere, edit settings etc...
  • Moderator / Blogger (2 Grade), can access everywhere, but can only make and edit posts.
  • User (1 Grade), can access only on the website and not in MyAdmin.

To add permission to any user, type his email and select the grade on the right panel.
User email can be found on users info.



Bans users

On this page you can ban any user, by email or ip.
If you ban a user, he can't view the website anymore, if you ban the email, he can recreate another account (only if the theme permit login and registration).

If you ban a user by ip, if he try access with this ip he can login in.

To ban a user, type on the right panel, his ip or his email and next select how much you want ban him.



Info users

On this page you can retrieve any users information.
In the center you can view a tables with all the user information. If you need the user email or the user ip you can easily get them, for ban the user or promote the user.
