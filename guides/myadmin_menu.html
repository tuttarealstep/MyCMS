Menu

Add new page to the menu

For create a new menu you need to use the right panel.

  • Name: The menu name.
  • Select Page: If you click on "I want use a custom link", you can click one of displayed page, created with MyPage.
    Or you can type a custom link in the under text box and leave in the select box "I want use a custom link".
    You can use the "{@siteURL@}" tag in the personal link.
  • Icon: You can also select a icon for your menu.

Remove

For remove a menu simply click the red "Delete" text near the menu, on the left of the page.

Order

For order the menu, you can drag on top or down the selected menu.
