Publish the theme:

Go to the main folder of your theme

Now you can view all the files and folders made before.



Select all files and folders and make a zip file.

Call it like the folder, without spaces and all the letters lowercase.

Like you can see, this is what you see if you open the .zip file.



Now cut the .zip file and paste it where you have the "info.json" file.



Open the "info.json" file and fill "theme_zip_file_name".

Write the name of the zip file without extension.



Now you can upload the "info.json" and the .zip file into a free web hosting or your private hosting, and place it in a folder.
Then grab the url of the info.json and paste it in the theme page, in the admin panel.


If you want you can also upload manually your theme, go to the theme manage an click on choose file, then click the zip file and finally click upload.
